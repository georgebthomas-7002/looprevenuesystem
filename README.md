# Modern Website Boilerplate

A clean, modern website structure ready for development.

## Project Structure

```
├── src/                  # Source files
│   ├── css/             # Stylesheets
│   ├── js/              # JavaScript files
│   ├── images/          # Source images
│   └── fonts/           # Custom fonts
├── public/              # Public static files
├── assets/              # Compiled/optimized assets
│   ├── images/
│   ├── icons/
│   └── fonts/
├── tests/               # Test files
├── docs/                # Documentation
├── index.html           # Main HTML file
└── README.md            # This file
```

## Getting Started

1. Open `index.html` in your browser
2. Edit files in the `src/` directory
3. Add your styles to `src/css/`
4. Add your scripts to `src/js/`

## Development

- Place your development files in `src/`
- Optimized/production files go in `assets/` or `public/`
- Keep your code organized and modular
